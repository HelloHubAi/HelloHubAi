# HelloHub – AI Receptionist

Never miss a call again. 24/7 AI phone receptionist built with Next.js, Supabase, Stripe & Vapi.

## Features
- Instant call answering
- Natural human-like voice
- Book appointments automatically
- Accept payments over the phone
- 5-minute setup

## Quick Deploy
1. Click "Use this template" or fork
2. Deploy to Vercel
3. Add your env vars
4. Done!

More coming: call logs, custom scripts, team access, PayPal, etc.
